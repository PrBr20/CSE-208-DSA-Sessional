1. There are 4 cpp files in the folder naming : "List_interface.cpp", "AList.cpp", "LList.cpp", "Tnl.cpp".
2. The "List_interface.cpp" file contains some pure virtual function demonstrating the functionalities of List ADT.
3. The "AList.cpp"  and "LList.cpp" file includes the "List_interface.cpp" file for implementing those methods.
4. Finally, The "Tnl.cpp" file contains the TNL class which is implemented through the List ADTs made in the previous files and 
the main function for checking the List ADTs as well as the demonstration of the methods in the TNL class.
